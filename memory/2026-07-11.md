# 2026-07-11

## FamCloud Session Summary — Product Architecture & Strategy

### Product Definition
- **Privacy-first local AI assistant** that lives on a GPU rig in the customer's home
- All data, all inference, all memory stays local. Nothing leaves the house.
- Agent has its OWN identity (phone number, email, WhatsApp, etc.) — family members message the agent
- Agent also connects to family's accounts (Google OAuth, etc.) for context

### Hardware Per Unit
- **Mini-ITX build**: GPU (RTX 4060 Ti 16GB or better), CPU (Ryzen 5 5600X), 32GB DDR5, 500GB NVMe, Mini-ITX case (NR200), SFX PSU 750W
- Target cost: ~$1,120/unit
- Ubuntu + OpenClaw pre-installed
- Self-serve onboarding: plug in → open famcloud.local:18789/setup → wizard

### Agent-Native Tool Stack (Final)
- **Phone/iMessage/SMS/Voice routing: AgentPhone** (agentphone.ai) — agent-owned number, iMessage-enabled, one API
- **Voice processing: Pipecat** (self-hosted on rig's GPU) — Whisper STT, local LLM, local TTS. $0/min. We own the stack.
- **WhatsApp: Baileys** (self-hosted on rig, QR pair). Phase 1 FREE. Migrate to AgentPhone when available.
- **Telegram**: @BotFather bot (FREE)
- **Discord**: Developer Portal bot (FREE)
- **Slack**: Slack API app (FREE)
- **Email**: famcloud.ai domain + Cloudflare Email Routing (FREE, $15/year for domain)
- **Google context**: OAuth for Gmail/Calendar/Drive (FREE, customer grants)
- **AgentPhone handles**: phone number provisioning, iMessage relay, SMS, voice call routing (inbound audio → Pipecat via webhook → TTS audio back)

### What's NOT in the Stack
- ❌ Telnyx (rejected — AgentPhone is more agent-native)
- ❌ Vapi/Bland/Retell (rejected — Pipecat is free, self-hosted, we own it)
- ❌ OpenRouter API (rejected — ALL inference runs LOCAL on customer's GPU)
- ❌ Tailscale (rejected — SSH via bastion is sufficient)
- ❌ Customer Mac for iMessage (rejected — AgentPhone provides iMessage-enabled numbers)
- ❌ BlueBubbles (rejected — same reason)

### Per-Household Monthly Cost: $2-6
- AgentPhone number (phone + iMessage + SMS + voice routing): $1-5/month
- famcloud.ai domain: $1.25/month
- All other channels: FREE (self-hosted)
- Local AI inference: FREE (runs on customer's GPU)
- Local voice (Pipecat): FREE (runs on customer's GPU)
- Shared infra (update server, bastion, Stripe): $18-25/month TOTAL, shared across ALL households

### Pricing Tiers
- **FamCloud**: $49/mo + $500 setup (1 channel, basic memory, 2 members)
- **FamCloud Family**: $99/mo + $800 setup (3 channels, extended memory, 4 members, parental controls)
- **FamCloud Premium**: $149/mo + $800 setup (ALL channels, unlimited memory, unlimited members, voice calls, priority support)

### Self-Serve Onboarding Flow
1. Customer plugs in rig → boots → bootstrap runs
2. Opens famcloud.local:18789/setup
3. Creates household account
4. We auto-provision: AgentPhone number + email alias (one click)
5. Customer connects: WhatsApp QR, Telegram token, Discord bot, Slack app, Google OAuth
6. Agent live on all channels
7. Bootstrap handles: NVIDIA drivers, Docker, OpenClaw, models, Pipecat, health monitor, update cron
